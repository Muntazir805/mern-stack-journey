CSS Units & Measurements

1. Absolute Unit px (pixel)

This text uses px unit.

Padding:20px | Font-size:20px | border:2px

2. Relative Units

% (Percentage)

width:80%

Relative to the parent element.

em

Font-size:2em

Relative to the font-size of the parent element.

rem

Font-size:2rem

Relative to the font-size of the root (html) element.

vw (Viewport Width)

Width:50vw

1vw = 1% of the viewport width.

vh (Viewport Height)

Height:50vh

1vh = 1% of the viewport height.

vmin

Width:20vmin

Relative to the smaller dimension of viewport (width or height).

vmax

Width:20vmax

Relative to the larger dimension of viewport (width or height).

3. Funtions Using Units

calc()

Width:calc(100% - 40px)

Full width minus 40px

min()

Width:min(600px , 80%)

Takes the smaller value

max()

Width:max(300px , 50%)

Takes the larger value

clamp()

font-size:clamp(16px , 2vw , 32px)

Responsive font size
